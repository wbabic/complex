* complex
A cross platform library for complex numbers, geometry and transforms

** Usage
run cljs test using phantomjs
#+BEGIN_SRC shell
  lein cleantest
#+END_SRC

run clojure test from the repl
#+BEGIN_SRC shell
  lein repl
#+END_SRC

#+BEGIN_SRC clojure
  (require 'complex.test-runner)
  (complex.test-runner/runner)
#+END_SRC
